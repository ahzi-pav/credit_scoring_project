## Credit Scoring Project

Overview:

This is a credit scoring project that features main steps learned from the Machine Learning Zoomcamp lessons until 6th week. 
The project includes basic data preparation, exploratory analysis, and machine learning.

Model used: XGBoost

How to use?

Execute the run_service.bat to run the model service. Please note that you need an active docker running.

The notebook used to train the model is available, if you want to simulate the training, please note that you need the packages installed in your environment.

After executing the run_service.bat, you can simulate the post request through the simulate-post-request.ipynb, there is an example borrower profile defined there.
